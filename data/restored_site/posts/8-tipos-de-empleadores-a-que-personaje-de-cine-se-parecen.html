8 Tipos de Empleadores. ¿A qué Personaje de Cine se parecen?

n

Es importante que conozcamos los distintos proveedores de selección con los que nos podemos encontrar a la hora de externalizar. De este modo estaremos más preparados y sabremos lidiar con la situación de manera adecuada.

n

A continuación te mostramos los tipos de empleadores que hay en función de su carácter, manera de hacer y con qué personaje de cine los podemos relacionar.

n

1) El que pasa de ti

n

Probablemente se trate de un adicto al trabajo que no entienda el significado de “tiempo libre”. Este tipo de empleadordifícilmente encontrará unos minutos para llamarte por teléfono o enviarte un simple correo.

n

 

n

 

n
n

Gordon Gekko (Michael Douglas) en Wall Street: el dinero nunca duerme. 

n

n

2) El narcisista

n

Este tipo de empleador se asegurará de hacerte ver el esfuerzo que le supone quedar contigo y lo afortunado que eres por deleitarte con su presencia. Le gusta sentirse importante y a veces se atribuye logros ajenos.

n

 

n

 

n
n

Stan Smith en la serie de TV American Dad. 

n

 

n

3) El “quisquilloso”

n

Sufre de una falta de confianza en aquellos que trabajan con él y necesita supervisar constantemente tus acciones. Te llama constantemente pidiéndote detalles y cree que todo lo que haces es susceptible de mejora.

n

 

n

 

n
n

Jonah Jameson (Cristopher MacDonald) en Spiderman. 

n

4) El “amigo”

n

Se esfuerza en llevarse bien contigo y crear una relación de confianza. Es empático, cercano y tratará de explicarte sus decisiones personalmente y hacerte partícipe de todo cuanto pueda ayudarte a llevar a cabo la búsqueda

n

 

n

 

n
n

Nelson Mandela (Morgan Freeman) en Invictus. 

n

 

n

5) El abusivo

n

Este tipo de empleador hará que des lo máximo de ti mismo, pero un exceso puede ser agotador. Deberás trabajar constante y rigurosamente hasta que le convenzas de que ha exprimido al máximo tus habilidades para el beneficio de la compañía.

n

 

n

 

n
n

Miranda Priestly (Meryl Streep) en El Diablo viste de Prada. 

n

 

n

 

n

 

n

6) El autoritario

n

Se comunica contigo unidireccionalmente y encaja a la perfección con la frase “hazlo como yo lo digo”. Probablemente no escuche tus propuestas, tomará las decisiones de forma restrictiva y no sentirá lástima por nadie que no cumpla con sus expectativas.

n

 

n
n

Sargento Hartman (R. Lee Ermey) en La Chaqueta Metálica. 

n

 

n

 

n

7) El que muestra interés pero los departamentos no le hacen caso

n

Se trata de una persona atenta, paciente y constante. Te escuchará y tratará de orientarte siempre que pueda. Sin embargo, los departamentos de Recursos Humanos no le hacen caso por lo que difícilmente luchará por tus propuestas de cambio.

n

 

n

 

n
n

Peter Parker (Tobey Maguire) en Spiderman.

n

 

n

 

n

8) El que va con calma por la vida

n

Lidiar con un empleador lento no siempre es fácil, pues te hará ralentizar el proceso más de lo que te gustaría. Si ves que tarda mucho en revisar los currículums u otras tareas que no te permiten avanzar, probablemente estés ante una persona poco organizada o con demasiadas cosas en la cabeza.

n

 

n

 

n
n

Jimmy Stark (James Dean) en Rebelde sin causa. 

n

 

n

 

n

Son muchas las ventajas que te puede aportar la externalización de tus procesos de selección, sin embargo, es normal que antes de dar el paso quieras informarte más acerca de la práctica.

n
