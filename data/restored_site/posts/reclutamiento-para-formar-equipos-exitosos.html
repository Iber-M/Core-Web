Reclutamiento para formar equipos exitosos

Si bien es cierto que el éxito de una empresa tiene como base el talento humano, en la actualidad el área de Recursos Humanos ha evolucionado y se ha convertido en un puesto estratégico con múltiples habilidades que permiten mejores prácticas, ser vanguardistas y saber formar equipos ideales 

n

Hemos encontrado 4 actividades que cualquier reclutador debe hacer, así como las habilidades que debe desarrollar constantemente para atenderlas con éxito:

n
    n
  1. Reclutamiento
    2. n
  2. Capacitación
    3. n
  3. Seguimiento
    4. n
  4. Motivación
    5. n
n

En esta colaboración compartiré algunos tips y características de la primera, para tener las suficientes herramientas al momento de contratar colaboradores y llegar a nuestro objetivo.

n

Pensemos en el reclutamiento como un proceso de ventas. Es primordial que aquellos que se dedican a encontrar y entrevistar los talentos atraídos por una oferta laboral sepan vender, pero también tengan identificado el perfil para el cual están contratando.

n

Muchas veces encontramos que, con tal de contratar rápido a alguien (por la urgencia de llenar el puesto por exigencia del jefe) ni siquiera nos detenemos a leer cuidadosamente el currículum de los candidatos, hacemos la tan famosa “prueba del espejo”, esto es: ponemos un espejo bajo su nariz, si lo empaña, lo contratamos.

n

Por lo tanto, será sumamente importante que tenga el perfil correcto para el puesto correcto. Identifiquemos y describamos las piezas clave que conformen un equipo de trabajo exitoso con las siguientes preguntas:

n
    n
  • ¿Qué tipo de persona estoy buscando?
    • n
  • ¿Qué experiencia laboral y trayectoria académica necesitamos?
    • n
  • ¿Qué habilidades humanísticas necesitamos?
    • n
  • ¿Qué características tienen nuestros colaboradores más exitosos?
    • n
  • ¿Dónde estamos buscando a los prospectos?
    • n
n

Por otro lado, si vemos al reclutamiento como un proceso de venta, debemos llevarlo de la mano, con la técnica de ventas correcta, siguiendo un orden lógico de pasos para atender las necesidades del prospecto, pero también las de nuestro corporativo.

n

Pregúntate lo siguiente:

n
    n
  • ¿Conozco a la empresa para la cual colaboro?
    • n
  • ¿Sé lo que ofrecemos?
    • n
  • ¿Sé lo que necesitamos?
    • n
  • ¿Qué ventajas tendría trabajar con nosotros 
    • n
n

Recuerda que tenemos que ser “atractivos” a los ojos de aquellos que se acercan a solicitarnos empleo y evitar ciertos pensamientos para no tener “malas contrataciones”:

n
    n
  • “Les hacemos un favor al darles empleo”.
    • n
  • “Nos hacen un favor al aceptar nuestra propuesta”.
    • n
n

Si logramos coincidir nuestra oferta de negocios con las expectativas de nuestro candidato, tendremos un colaborador con la camiseta puesta.

n

Es así como estos consejos pueden resultarnos muy útiles para estructurar nuestros procesos reclutamiento y conseguir al candidato ideal, y, por lo tanto, formar al mejor equipo de trabajo, tal cual es el objetivo.

n

Piensa, reflexiona y actúa
